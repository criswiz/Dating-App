Dating App - Mobile (Flutter)

Quickstart

1. From the `mobile_app/` folder, run:

```bash
flutter pub get
flutter run
```

Files of interest
- `lib/main.dart` - app entry
- `lib/screens/login.dart` - login form
- `lib/screens/signup.dart` - signup form
- `lib/screens/swipe.dart` - prototype swipe UI

This is a minimal skeleton. Next steps: implement secure token storage, image upload, swipe card visuals, and push notifications.
