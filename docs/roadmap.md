# Dating App Roadmap

## Strategy

- Build with a no-cost deterministic matcher first.
- Defer paid AI features until KPI and revenue gates are met.
- Prioritize safety/trust and conversation quality after core matching.

## Active Phases

- [x] Phase 0: Tracking and test infrastructure
- [x] Phase 1: Rule-based matchmaking
- [x] Phase 2: Core engagement (likes, matches, chat)
- [x] Phase 3: Safety and trust
- [x] Phase 4: AI readiness gate

## Current Sprint Focus

- Stabilize production hardening (migrations, observability, deployment automation).
- Add richer conversation quality features with guarded rollout flags.
- Expand moderation tooling and admin workflow UX.
- Add KPI dashboards and periodic checkpoint reviews.
